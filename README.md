# UNICC AI Safety Lab — Council of Experts System

## Overview

This project implements a **Council of Experts AI Safety Evaluation System** for the UNICC AI Safety Lab Capstone.

The system evaluates an AI application using three independent expert modules:

- Governance Expert
- Threat Expert
- Behavioral Expert

These expert outputs are combined through a deterministic arbitration layer to produce a final council decision.

Final decisions are aligned with evaluation rubric:

- APPROVE
- REVIEW
- REJECT
